# Android App - Custom Application Icon

## Deskripsi
Project ini merupakan tugas Android yang bertujuan untuk mengubah ikon (launcher icon) aplikasi Android Studio. Perubahan ikon dilakukan menggunakan fitur **Image Asset Studio** sehingga aplikasi memiliki tampilan ikon baru pada launcher perangkat Android.

## Tujuan
- Memahami struktur resource pada Android.
- Mempelajari cara mengganti launcher icon aplikasi.
- Menggunakan fitur Image Asset Studio di Android Studio.

## Tools yang Digunakan
- Android Studio
- Kotlin
- Android SDK
- Image Asset Studio

## Langkah-langkah Mengubah Icon
1. Buka project di Android Studio.
2. Klik kanan pada folder **app**.
3. Pilih **New → Image Asset**.
4. Pilih jenis ikon **Launcher Icons (Adaptive and Legacy)**.
5. Masukkan gambar yang akan dijadikan ikon aplikasi.
6. Atur ukuran dan posisi ikon sesuai kebutuhan.
7. Klik **Next** kemudian **Finish**.
8. Jalankan aplikasi pada emulator atau perangkat Android.

## Hasil
Ikon aplikasi berhasil diganti dan akan tampil pada launcher perangkat Android dengan ikon baru yang telah dipilih.

## Struktur Folder yang Berubah
```
app/
└── src/
    └── main/
        └── res/
            ├── mipmap-anydpi-v26/
            ├── mipmap-hdpi/
            ├── mipmap-mdpi/
            ├── mipmap-xhdpi/
            ├── mipmap-xxhdpi/
            └── mipmap-xxxhdpi/
```
